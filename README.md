# 🐉 Dragon Slider — Portfolio Banner

An interactive **3D carousel banner** built with **HTML, CSS and vanilla JavaScript**.

---

## ✨ Features

### 🌀 3D Rotating Carousel
10 cards orbit around a central figure using pure CSS with  
`preserve-3d` and `rotateY` transforms.

### ⌨️ Typewriter Effect
The name animates in a **write → pause → erase → repeat loop**  
every ~6 seconds.

### 🐲 Dragon Info Panel
Clicking any card slides in a **detail panel** with:

- Dragon name
- Element
- Origin
- Description
- Animated stat bars

### 📱 Fully Responsive
The layout, carousel radius and typography adapt to:

- Desktop
- Tablet
- Mobile

### ✨ Decorative Details
Includes several visual enhancements:

- Subtle grid background
- Sparkle SVG effects
- `mix-blend-mode` model integration

---

## 🛠️ Built With

- **HTML5**
- **CSS3**
  - Custom properties
  - `transform-style: preserve-3d`
  - `clamp()`
  - Media queries
- **Vanilla JavaScript**
  - No frameworks
  - No external dependencies
